Elais Couture

Description du projet

Site de type blog / vitrine sur le thème de la couture réalisé pour un particulier.
Le site comprend une partie front-end ainsi qu'une partie back-end afin de communiquer avec la base de données.

Cette base de données comporte 5 tables afin de stocker les articles et créations, ainsi que leurs différentes images et enfin une table permettant de stocker les identifiants des administrateurs du site.

Ces derniers ont accés à une partie administration protégée par login afin d'ajouter, modifier ou bien supprimer du contenu.

J'ai crée l'ensemble de l'interface du site, ainsi que le logo.

Technologies / Outils Utilisés

html, scss, javascript, php, mysql, phpmyadmin, adobe illustrator
Voir le site : http://elais-couture.com/

Quiz Casting de film

Description du projet

Étant passionné de quiz et de cinéma j'ai voulu créer un site qui propose de deviner un film à partir de son casting.

Le jeu consiste donc en l'affichage des cinq acteurs et actrices principaux d'un film.

Le joueur clique sur un bouton chaque fois qu'il souhaite afficher un nouveau nom d'acteur. Une fois les 5 acteurs affichés le joueur peut afficher le titre et l'affiche du film en question et voir s'il avait bien deviné.

Afin d'obtenir les informations sur les films (listes d'acteurs, titres français, affiches) l'API de la base de données TMDB est utilisée.

Pour que la difficulté ne soit pas trop élévée, j'ai choisi de filtrer les films en selectionnant seulement les 1000 plus populaires. Le site est hébérgé sur Github Pages.

Technologies / Outils Utilisés

HTML, CSS, SCSS, JAVASCRIPT, API, GITHUB
Lien du site : https://remilagardere.github.io/Quiz-Casting-Film/
Lien du Github : https://github.com/RemiLagardere/Quiz-Casting-Film

To Do List

Description du projet

To Do List permettant à l'utilisateur de créer des tâches, et de les laisser afficher sous forme de Post-It.

Une fois la tâche terminée il a le choix de la rayer ou bien de la supprimer.

La fonction LocalStorage est utilisée pour stocker les données directement dans le navigateur. Ce qui permet de conserver les tâches même après redémarrage du navigateur ou de l'appareil utilisé.

Technologies / Outils Utilisés

HTML, CSS, SCSS, JAVASCRIPT, LOCALSTORAGE, GITHUB
Lien du site : https://remilagardere.github.io/Quiz-Casting-Film/
Lien du Github : https://github.com/RemiLagardere/Quiz-Casting-Film
